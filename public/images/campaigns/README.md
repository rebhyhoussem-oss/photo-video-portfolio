# Brand Campaigns — Drop Your Files Here

Upload your campaign thumbnail images directly into this folder using GitHub's
web UI ("Add file" → "Upload files").

## Exact filenames expected (name your files exactly like this):

- `campaign-01.jpg`
- `campaign-02.jpg`
- `campaign-03.jpg`

## Tips

- Recommended size: 1600×900 (widescreen 16:9)
- Formats supported: `.jpg`, `.png`, `.webp` (`.webp` loads fastest)
- If your file is `.png` or `.webp` instead of `.jpg`, update the matching
  `src` path in `src/data/media.ts` to match (e.g. `campaign-01.png`)
- Want more or fewer campaigns? Add/remove entries in `src/data/media.ts` —
  the layout adjusts automatically
- No layout code needs to change — just replace these files and push
